# Contributing to CRM Local MVP

Спасибо за интерес к проекту! Ниже минимальные правила и шаги для внесения изменений.

1. Форкните репозиторий и создайте ветку по шаблону `feature/описание` или `fix/описание`.
2. Следуйте стилю кода: форматирование `black` и линтинг `ruff` рекомендуется.
3. Перед коммитом запустите тесты (если добавлены) и линтер:

```powershell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
# optional: pip install ruff black
# запустить тесты
pytest -q
# проверить линтинг
ruff check .
```

4. Пишите ясные сообщения коммитов и оформляйте PR с описанием изменения и ссылками на задачи.
5. Для изменений конфигурации не храните секреты в репозитории — используйте `.env` и добавьте `.env.example`.

Checklist перед PR:
- Все тесты проходят
- Линтер не сообщает ошибок
- Обновлена документация при необходимости

Если нужна помощь с настройкой окружения или CI — откройте issue.
